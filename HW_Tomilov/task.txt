Выберите любой из трёх датасетов и постройте наиболее качественную модель для предсказания следующих переменных:
1) BodyFat для bodyfat_regression.csv
2) HeartDisease для heart_binaric.csv
3) Category для hepatisis_multiclass.csv

В папке находится описание каждого датасета в соответствующем файле.

В итоге у вас должен получиться проект с rmd-файлом, в котором будет написан весь код обучения и сохранения модели в файл формата rds. Пожалуйста, комментируйте каждый свой шаг. :)
